### govulncheck pre-commit hook (commit 145d768)

Added `govulncheck-on-gomod-change` hook to `.pre-commit-config.yaml` per user instruction, along with ADR-0012 documentation updates:

- Hook triggers only on `go.mod` changes (too slow + network-dependent for every commit)
- Invokes `scripts/govulncheck.sh` — a workspace-aware wrapper that iterates modules, reads toolchain version from `go.work`, and prepends `~/sdk/goX.Y.Z/bin` to PATH so stdlib scans match the actual build toolchain

- ADR-0012 updated: four-layer CVE coverage table, commit-time rationale note, corrected Go hooks section (was still showing `dnephin/pre-commit-golang`; replaced with actual local hooks last session), govulncheck workspace-aware script rationale
